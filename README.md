# @cozanet/shared

Shared utilities, interfaces, system types, and configuration for CozanetOS.

## Structure

- `src/Types/system.ts`: Core type systems.
- `src/Interfaces/`: Interfaces for engines, agents, and stores.
- `src/Schemas/`: Dynamic validations schemas.
- `src/Constants/`: Shared configurations, ID listings, and types.
- `src/Utilities/`: Shared helper facilities (e.g. logger, ID generator).
- `src/SDK/`: Lightweight client implementation for external access.
